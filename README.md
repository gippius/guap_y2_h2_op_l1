# Лабораторная работа №1

Тема: "Классы, определение методов класса, права доступа".

В проекте реализован вариант 4: класс клетки шахматной доски с `private`-полями, `set`/`get`-методами и проверкой цвета клетки.

## Сборка

```powershell
cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Запуск

```powershell
.\build\Release\lab1_chess_cell.exe
```
